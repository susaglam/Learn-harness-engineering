# Code of Conduct

## Our pledge

We want this project to be a welcoming place to learn harness engineering,
regardless of experience level, background, or native language (the course ships
in English and Türkçe precisely so more people can take part).

## Standards

- Be kind and constructive. Assume good faith; critique ideas, not people.
- Keep feedback specific and actionable — the same bar the course holds for evals.
- No harassment, discrimination, or demeaning language.

## Scope

This applies in all project spaces (issues, pull requests, discussions) and when
representing the project publicly.

## Enforcement

Report unacceptable behavior by opening a confidential issue or contacting the
maintainer. Maintainers may edit, remove, or reject contributions and comments
that violate this code, and may ban repeat offenders.

This is a lightweight adaptation of the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/).
